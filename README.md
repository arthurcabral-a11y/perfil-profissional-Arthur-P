# perfil-profissional-Arthur-P
Projetos acadêmicos e pessoais, com oobjetivo de compor um perfil profissional robusto, as melhores práticas de versionamento, colaboração e compartilhamento decódigo, e como integrar este repositório com outras plataformas profissionais, como oLinkedIn.
Meu  atual  projeto tem como objetivo desenvolver um repositório profissional no GitHub para mim,  sou o estudante Arthur Parente Cabral, reunindo projetos acadêmicos e pessoais de forma organizada, documentada e versionada. Essas áreas possuem forte relação entre si, principalmente por utilizarem conceitos matemáticos e lógicos para resolução de problemas computacionais. A matemática da computação, por exemplo, auxilia na compreensão de algoritmos, sistemas binários e operações lógicas, enquanto a lógica de programação desenvolve a capacidade técnica necessária para construir softwares e sistemas computacionais.
Além disso, a Engenharia de Software complementa esse aprendizado ao introduzir práticas de organização, planejamento, documentação e manutenção de projetos, tornando o desenvolvimento mais profissional e estruturado.
